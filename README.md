# 📸 Photo Gallery Telegram Bot

Telegram бот для просмотра и загрузки фотографий и видео. Бот позволяет пользователям получать случайные фотографии, просматривать альбомы и загружать контент (только для администратора).

## ✨ Возможности

- 🎲 **Случайные фотографии** - получение одного случайного фото
- 📸 **Альбомы** - получение 10 или 30 случайных фотографий за раз
- 📤 **Загрузка контента** - возможность загружать фото и видео (только для администратора)
- 🎨 **Красивый интерфейс** - удобные кнопки и стикеры
- 📊 **Логирование** - подробное логирование действий пользователей

## 🚀 Команды

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу с ботом |
| `/photo` | Получить одно случайное фото |
| `/photo_10` | Получить 10 случайных фотографий |
| `/photoX` | Получить 30 случайных фотографий |
| `/download_photo` | Информация о загрузке контента |
| `/developer` | Информация о разработчике |
| `/help` | Показать список команд |

Также доступна кнопка **"Хочу фото🙀"** в меню для быстрого получения случайного фото.

## 📋 Требования

- Python 3.7 или выше
- Библиотека `pyTelegramBotAPI`
- Токен Telegram бота

🐛 Решение проблем
Бот не отвечает
Проверьте, правильно ли указан токен в TOKEN.py

Убедитесь, что интернет-соединение стабильно

Проверьте логи на наличие ошибок

Фотографии не отправляются
Проверьте файл photo.txt - в нем должны быть валидные URL-адреса

Убедитесь, что URL-адреса ведут к изображениям

Проверьте права доступа к файлам

Ошибка при загрузке фото
Убедитесь, что папки photo/ и video/ существуют и доступны для записи

Проверьте права администратора
